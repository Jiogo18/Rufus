5. Pour les importer dans MySQL
	1. Utiliser MySQLWorkbench
	2. Clic droit sur le nom rufus, choisir Table Data Import Wizard
	3. Choisir le fichier à importer
	4. Choisir un nom de table, cocher create new table, cocher drop table if exists 
	5. Garder les formats par défaut
	6. Une fois la table créée, modifier les formats par un alter table: date = date code = varchar(7), nom = varchar(256)
	7. Exécuter le requête "create table rufus.CCAMXX as (SELECT distinct code , nom FROM rufus.r_acte);" pour éliminer tous les doublons de R_ACTE
